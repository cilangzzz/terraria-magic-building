## 2. 工具实现详解

### 2.1 search_tiles (混合检索)

**执行流程：**
```
输入: style="medieval", category="brick", biome="forest"
    ↓
Step1: SQL精确过滤
    kb.Tiles.SearchTiles(null, "brick", "forest")
    → 候选集: [GrayBrick, StoneSlab, Stone, Wood, ...]
    ↓
Step2: 向量语义排序
    kb.Vectors.SearchTilesSemantic(candidates, "medieval", 20)
    → 计算 style_vector 与每个tile_vector 的余弦相似度
    → 按相似度降序排序，取Top-20
    ↓
输出: {
  "tiles": [{"id":4,"name":"GrayBrick","display_name":"灰砖"},...],
  "total_count": 20,
  "vector_search_used": true
}
```

**代码实现 (AIAgentService.cs:582-605):**
```csharp
private ToolResult SearchTiles(string style, string category, string biome, KnowledgeBaseManager kb)
{
    // Step1: SQL精确过滤 (category, biome)
    var candidates = kb.Tiles.SearchTiles(null, category, biome).ToList();

    // Step2: 向量语义排序 (style)
    if (!string.IsNullOrEmpty(style) && kb.Vectors.IsInitialized)
    {
        candidates = kb.Vectors.SearchTilesSemantic(candidates, style, 20);
    }
    else
    {
        candidates = candidates.Take(20).ToList();
    }

    return new ToolResult {
        IsError = false,
        Content = JsonConvert.SerializeObject(new {
            tiles, total_count, search_criteria, vector_search_used
        })
    };
}
```

private ToolResult GetStyleTemplate(string style, string buildingType, KnowledgeBaseManager kb)
{
    var template = kb.Styles.GetTemplate(style, buildingType);
    if (template == null)
        return new ToolResult { IsError = true, Content = "{\"error\": \"未找到风格\"}" };

    var paintScheme = kb.Tiles.GetPaintRecommendation(style);
    return new ToolResult {
        Content = JsonConvert.SerializeObject(new {
            style, name, display_name, description,
            paint_scheme = new { primary, shadow }
        })
    };
}

## 5. 向量检索实现

### 5.1 VectorKnowledgeBase核心方法

```csharp
// 余弦相似度计算
public float CosineSimilarity(float[] a, float[] b)
{
    float dot = 0, magA = 0, magB = 0;
    for (int i = 0; i < a.Length; i++)
    {
        dot += a[i] * b[i];
        magA += a[i] * a[i];
        magB += b[i] * b[i];
    }
    return dot / sqrt(magA * magB);
}

// 混合检索
public List<TileInfo> SearchTilesSemantic(List<TileInfo> candidates, string style, int topK)
{
    var styleVector = GetStyleVector(style);
    return candidates
        .Select(t => (tile: t, score: CosineSimilarity(GetTileVector(t.id), styleVector)))
        .OrderByDescending(s => s.score)
        .Take(topK)
        .Select(s => s.tile)
        .ToList();
}
```
