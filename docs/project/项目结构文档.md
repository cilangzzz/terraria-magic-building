## 核心模块说明

### Core/API - API服务层

提供与各种AI API的统一接口，支持OpenAI、Claude、DeepSeek、DashScope等格式。

### Core/Agents - Agent服务层

实现真Agent架构，包含工具调用、多轮对话、自主决策能力。

**工具列表**:
- SearchMaterialsTool - 搜索建筑材料
- SearchBuildingTemplatesTool - 搜索建筑模板
- GetMaterialRecommendationTool - 获取材料推荐
- GetTemplateDetailsTool - 获取模板详情
- GetBuildingSequenceTool - 获取建筑序列
- GenerateDesignRulesTool - 生成设计规则
- ValidateRequirementsTool - 验证需求

### Core/Building - 建筑执行层

将建筑设计转换为游戏世界中的实际建筑，支持油漆、斜坡、阴影效果。

### Core/KnowledgeBase - 知识库层

提供方块、墙壁、家具、风格等数据的查询和语义检索能力。

## 数据流程

```
用户描述 → AIAgentService
              ↓
         TrueAgentCore (工具调用循环)
              ↓
         ToolRegistry (分发工具调用)
              ↓
         KnowledgeBase/BuildingEntityBase
              ↓
         BuildingDesign
              ↓
         BuildingExecutor → 世界放置
```
