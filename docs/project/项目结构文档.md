# trab 模组项目结构文档

## 项目概述

**项目名称**: trab (AI Building Generator)
**作者**: cilang
**版本**: 0.3
**架构**: 构件级生成架构

**功能**: 通过 AI Agent 在泰拉瑞亚游戏中自动生成建筑

---

## 架构特点

### 构件级生成

采用多层次架构设计:

```
层次4: Complex (建筑群)    → 村庄、基地
层次3: Building (完整建筑) → 住宅、塔楼、商店、神庙
层次2: Composite (复合构件) → 房间、楼层
层次1: Atomic (原子构件)   → 屋顶、墙壁、装饰、基础
```

### 核心流程

```
用户描述 → TrueAgentCore
              ↓
         工具调用循环 (检索→理解→生成)
              ↓
         BuildingRules (精简设计规则)
              ↓
         ProceduralBuilder (程序化展开)
              ↓
         TEditSchDesign (完整方块数据)
              ↓
         BuildingExecutor → 世界放置
```

## 核心模块说明

### Core/Agents - Agent服务层

#### TrueAgentCore

真Agent核心实现，统一处理用户请求。

**工具调用流程**:
1. `search_buildings` - 检索建筑模板
2. `get_building_details` - 获取构件详情
3. `get_component_rules` - 获取生成规则
4. `get_style_materials` - 获取材料推荐
5. `generate_design_rules` - 生成设计规则JSON

#### Agent工具集

| 工具类 | 功能 |
|--------|------|
| BuildingTools | 建筑检索工具集 |
| SearchMaterialsTool | 搜索建筑材料 |
| GetMaterialRecommendationTool | 获取材料推荐 |
| GetTemplateDetailsTool | 获取模板详情 |
| GetBuildingSequenceTool | 获取建筑序列 |
| GenerateDesignRulesTool | 生成设计规则 |
| ValidateRequirementsTool | 验证需求 |

### Core/Building - 建筑执行层

#### ProceduralBuilder

程序化建筑生成器，**核心模块**。

将精简的设计规则（BuildingRules）展开为完整方块数据（TEditSchDesign）。

**生成流程**:
1. 初始化网格
2. 解析材料调色板
3. 生成框架
4. 生成墙壁
5. 生成楼层
6. 生成屋顶
7. 生成房间
8. 放置装饰

### Core/KnowledgeBase - 知识库层

#### ComponentKnowledgeBase

构件级建筑知识库，支持多层次架构。

**数据存储**:
- 原子构件定义
- 建筑实体
- 风格材料映射
- 建筑向量（语义检索）

### Data - 数据层

#### BuildingRules

设计规则数据结构，AI输出格式。

```json
{
  "name": "中式住宅",
  "width": 20,
  "height": 15,
  "style": "asian",
  "structure": {
    "frame": { "material": "primary" },
    "walls": { "outer_material": "primary" },
    "floors": [{ "level": 1, "thickness": 1 }],
    "roof": { "shape": "pagoda", "tier_count": 2 }
  },
  "materials": {
    "primary": { "tile_id": 179, "wall_id": 172 }
  }
}
```

## 数据流程

### 完整生成流程

```
用户输入: "一座中式风格的宝塔"

TrueAgentCore:
  第1轮: search_buildings({ style: "asian", building_type: "tower" })
  → 返回: [{ id: "chinese_pagoda_001", ... }]

  第2轮: get_building_details({ building_id: "chinese_pagoda_001" })
  → 返回: { components: ["roof_pagoda", "wall_outer", ...] }

  第3轮: get_style_materials({ style: "asian" })
  → 返回: { primary: { tile_id: 179, wall_id: 172 }, ... }

  第4轮: generate_design_rules({ style: "asian", width: 15, height: 25 })
  → 返回: BuildingRules JSON

ProceduralBuilder:
  GenerateFromRules(BuildingRules)
  → 展开为 TEditSchDesign

BuildingExecutor:
  BuildAtLocation(TEditSchDesign, x, y)
  → 在世界放置建筑
```
