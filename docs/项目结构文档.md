  - `shop` - 商店建筑

**文件**: [Config/AIBuildingConfig.cs](../Config/AIBuildingConfig.cs)

**配置项**:

| 配置名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|

**支持的 AI 服务**:
- `DeepSeek` - DeepSeek API (Anthropic 兼容端点)
- `Claude` - Anthropic Claude API
- `OpenAI` - OpenAI GPT API
- `DashScope` - 阿里云 DashScope API
- `Custom` - 自定义 API 端点

### 4. 核心模块 (`Core/`)

#### BuildingExecutor.cs

**文件**: [Core/BuildingExecutor.cs](../Core/BuildingExecutor.cs)

**功能**:
- 解析 AI 返回的 JSON 建筑数据
- 在游戏世界中放置方块、墙壁、家具

**核心方法**:

| 方法 | 功能 |
|------|------|
| `ParseDesign(string json)` | 解析 JSON 为 BuildingDesign 对象 |
| `BuildAtLocation(design, x, y, player)` | 在指定位置生成建筑 |

**支持的方块类型**:
- Stone, Dirt, Wood, Grass, GrayBrick, GoldBrick, SilverBrick
- CopperBrick, IronBrick, StoneSlab, Sandstone, Glass, SnowBlock
- Marble, Granite, Obsidian, Platform, Brick, WorkBench

**支持的墙壁类型**:
- StoneWall, DirtWall, WoodWall, GrayBrickWall, GoldBrickWall
- SilverBrickWall, CopperBrickWall, IronBrickWall, StoneSlabWall
- SandstoneWall, GlassWall, MarbleWall, GraniteWall, SnowWall, BrickWall

**支持的家具类型**:
- WorkBench, Table, Chair, Bed, Chest, Furnace, Anvil
- Piano, Dresser, Bathtub

#### AIApiService.cs

**文件**: [Core/AIApiService.cs](../Core/AIApiService.cs)

**功能**:
- 封装 AI API HTTP 请求
- 处理不同 API 格式 (Anthropic/OpenAI)
- 提供系统提示词配置

**系统提示词**:
```
你是泰拉瑞亚建筑设计助手。返回JSON格式建筑设计：
{
  "name": "建筑名称",
  "width": 宽度,
  "height": 高度,
  "tiles": [...],
  "walls": [...],
  "furniture": [...],
  "doors": [...],
  "lightSources": [...]
}
```

**请求流程**:
1. 构建 JSON 请求体
2. 发送 POST 请求到 API 端点
3. 解析响应并提取文本内容
4. 返回 AI 生成的文本

## 工作流程

### 建筑生成流程

```
1. 用户输入建筑描述
   └─ 方式A: /aibuild <描述>
   └─ 方式B: UI 面板 (按 G)

2. 发送请求到 AI API
   └─ AIApiService.SendChatRequestAsync()
   └─ 使用异步 Task.Run 避免阻塞游戏

3. AI 返回 JSON 建筑设计
   └─ ExtractJsonFromResponse() 提取 JSON

4. 解析建筑数据
   └─ BuildingExecutor.ParseDesign()
   └─ JsonConvert.DeserializeObject<BuildingDesign>

5. 在游戏世界生成建筑
   └─ BuildingExecutor.BuildAtLocation()
   ├─ 清空区域
   ├─ 放置墙壁
   ├─ 放置方块
   ├─ 放置门
   ├─ 放置家具
   ├─ 放置光源
   └─ 同步到客户端 (多人模式)

6. 刷新世界显示
   └─ WorldGen.SquareTileFrame()
   └─ NetMessage.SendTileSquare()
```

## 技术要点

### 异步请求处理

由于游戏运行在主线程，API 请求必须异步执行：

```csharp
Task.Run(async () => {
    var response = await service.SendChatRequestAsync(prompt, token);
    Main.QueueMainThreadAction(() => {
        // 切回主线程处理响应
        ProcessResponse(response);
    });
});
```

### 多人模式同步

建筑修改需要同步到所有客户端：

```csharp
if (Main.netMode != NetmodeID.SinglePlayer)
    NetMessage.SendTileSquare(-1, startX, startY, width, height);
```

### JSON 响应解析

AI 可能返回带 Markdown 的 JSON，需要提取：

```csharp
// 处理 ```json ``` 格式
int start = response.IndexOf("```json");
int end = response.IndexOf("```", start + 7);
// 或直接提取 {...} 大括号内容
int braceStart = response.IndexOf('{');
int braceEnd = response.LastIndexOf('}');
```
