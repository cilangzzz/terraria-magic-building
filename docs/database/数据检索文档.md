# 数据检索系统文档

## 概述

本模组实现了混合检索架构，结合SQL精确查询与向量语义匹配，为Agent工具提供智能数据检索能力。

```csharp
public List<TileInfo> SearchTiles(string style, string category, string biome)
{
    return _tiles.Where(t =>
        (style == null || t.styles.Contains(style)) &&
        (category == null || t.category == category) &&
        (biome == null || t.biome_match.Contains(biome))
    ).ToList();
}

// 分类查询
public List<TileInfo> GetTilesByCategory(string category)
{
    // category: brick, wood, luxury, light, door, natural, transparent, platform
    return _tiles.Where(t => t.category == category).ToList();
}
```

### 3.3 VectorKnowledgeBase

向量语义检索：

```csharp
public List<TileInfo> SearchTilesSemantic(List<TileInfo> candidates, string style, int topK)
{
    var styleVector = GetStyleVector(style);  // 获取风格向量

    return candidates
        .Select(t => (tile: t, score: CosineSimilarity(GetTileVector(t.id), styleVector)))
        .OrderByDescending(s => s.score)
        .Take(topK)
        .Select(s => s.tile)
        .ToList();
}

// 生物群落匹配
public List<TileInfo> SearchByBiome(List<TileInfo> candidates, string biome, int topK)
{
    var biomeVector = GetBiomeVector(biome);
    
    return candidates
        .Select(t => (tile: t, score: CosineSimilarity(GetTileVector(t.id), biomeVector)))
        .OrderByDescending(s => s.score)
        .Take(topK)
        .Select(s => s.tile)
        .ToList();
}

public float CosineSimilarity(float[] a, float[] b)
{
    float dot = 0, magA = 0, magB = 0;
    for (int i = 0; i < a.Length; i++)
    {
        dot += a[i] * b[i];
        magA += a[i] * a[i];
        magB += b[i] * b[i];
    }
    return dot / sqrt(magA * magB);
}
```

## 4. Agent工具

### 4.1 search_tiles

混合检索实现：

```csharp
private ToolResult SearchTiles(string style, string category, string biome, KnowledgeBaseManager kb)
{
    // Step1: SQL精确过滤
    var candidates = kb.Tiles.SearchTiles(null, category, biome);

    // Step2: 向量语义排序
    if (!string.IsNullOrEmpty(style) && kb.Vectors.IsInitialized)
    {
        candidates = kb.Vectors.SearchTilesSemantic(candidates, style, 20);
    }

    return new ToolResult { Content = Serialize(candidates) };
}
```

```csharp
private ToolResult ValidateHouse(HouseInfo house, KnowledgeBaseManager kb)
```

## 5. 向量生成

### 5.1 Python脚本 (Tools/python/generate_embeddings.py)

```python
# 使用SentenceTransformers (推荐)
from sentence_transformers import SentenceTransformer
model = SentenceTransformer('all-MiniLM-L6-v2')

# 方块向量
tile_text = f"{tile['name']} {tile['category']} {tile['styles']} {tile['description']}"
embedding = model.encode(tile_text)

# 简化方案 (无需依赖)
def generate_simple_vector(text, dim=384):
    vector = [0.0] * dim
    words = text.lower().split()
    for word in words:
        idx = hash(word) % dim
        vector[idx] += 1.0 / len(words)
    return normalize(vector)
```

### 5.3 方块向量生成示例

```python
def generate_tile_embedding(tile):
    """为单个方块生成向量"""
    # 组合多个字段作为向量文本
    text_parts = [
        tile['name'],
        tile['display_name'],
        tile['category'],
        ' '.join(json.loads(tile['styles'])),
        ' '.join(json.loads(tile['biome_match'])),
        tile['description']
    ]
    text = ' '.join([p for p in text_parts if p])
    
    # 使用模型生成向量
    embedding = model.encode(text)
    return embedding.tolist()
```

## 8. 性能优化

### 8.1 内存缓存

向量数据加载到内存字典，避免每次查询读取文件：

```csharp
private Dictionary<int, float[]> _tileEmbeddings;      // tile_id → vector
private Dictionary<string, float[]> _styleEmbeddings;  // style → vector
private Dictionary<string, float[]> _biomeEmbeddings;  // biome → vector
```

### 8.2 混合检索优势

| 纯SQL | 纯向量 | 混合检索 |
|-------|--------|----------|
| 精确但无语义 | 语义但慢 | 精确+语义 |
| O(n)扫描 | O(n)相似度计算 | SQL缩小候选 + 向量排序Top-K |

### 8.3 数据量优化

当前数据库经过筛选，只保留建筑相关数据：
- tiles: 35个（筛选掉矿石、植物、装饰等非建筑方块）
- walls: 23个（全部为建筑墙壁）
- 减少向量计算量，提升检索速度

## 10. 故障排查

| 问题 | 解决方案 |
|------|----------|
| 向量文件不存在 | 运行 `python Tools/python/generate_embeddings.py` |
| 向量检索返回空 | 检查向量是否正确加载，降低相似度阈值(默认0.3) |
| SQL查询慢 | 添加索引 `CREATE INDEX idx_tiles_category ON tiles(category)` |
| 风格匹配不准 | 补充风格向量关键词，或使用更高质量embedding模型 |
| 方块数据太少 | 运行 `python Tools/python/tile_crawler.py` 爬取更多数据 |
