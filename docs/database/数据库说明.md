# Terraria 建筑知识库数据库

## 数据文件

| 文件 | 大小 | 说明 |
|------|------|------|
| terraria_kb.db | 106KB | SQLite数据库文件 |
| terraria_kb_full.sql | 35KB | 完整SQL备份文件 |

## 数据库表结构

### 基础建筑数据

#### tiles (方块)
建筑方块数据，包含35个常用建筑方块。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | Tile ID |
| name | TEXT | 英文名称 |
| display_name | TEXT | 显示名称 |
| category | TEXT | 分类: brick/wood/luxury/light/door/natural/transparent/platform |
| styles | TEXT | 适用风格 (JSON数组) |
| biome_match | TEXT | 适用生物群落 (JSON数组) |
| paint_compatible | INTEGER | 是否支持油漆 |
| slope_compatible | INTEGER | 是否支持斜坡 |
| hardness | INTEGER | 硬度值 |
| light_emission | INTEGER | 光照强度 |
| description | TEXT | 描述 |

**方块分类统计**:
- brick: 灰砖、红砖、铜砖、铁砖、砂岩等
- wood: 木材、红木、针叶木、棕榈木、王朝木等
- luxury: 大理石、花岗岩、金砖、银砖、珍珠石等
- light: 火把、蜡烛、吊灯、立灯、篝火等
- door: 标准、活板门、大门
- transparent: 玻璃、冰块等
- natural: 石头、泥土、草地、雪块等
- platform: 各种平台

#### walls (墙壁)

| 字段 | 类型 | 说明 |
|------|------|------|

| display_name | TEXT | 显示名称 |
| category | TEXT | 分类 |

| paint_compatible | INTEGER | 是否支持油漆 |
| is_natural | INTEGER | 是否天然墙 |
| description | TEXT | 描述 |

| 28 | Shadow | 阴影 | depth |

#### slopes (斜坡)

### 建筑核心数据

#### style_templates (风格模板)
建筑风格模板数据，包含10种建筑风格。

| 风格 | 名称 | 主要方块 | 主要墙壁 |
|------|------|----------|----------|
| medieval | 中世纪风格 | GrayBrick, StoneSlab, Wood | GrayBrickWall, StoneWall |
| fantasy | 奇幻风格 | Pearlstone, Glass, GoldBrick | PearlstoneWall, GlassWall |
| steampunk | 蒸汽朋克风格 | CopperBrick, IronBrick | CopperBrickWall |
| natural | 自然风格 | Wood, Dirt, Stone | WoodWall, DirtWall |
| asian | 东方风格 | DynastyWood, Bamboo | DynastyWall |
| snow | 冰雪风格 | SnowBlock, IceBlock, BorealWood | SnowWall, IceWall |
| desert | 沙漠风格 | Sandstone, PalmWood | SandstoneWall |
| underground | 地下风格 | Stone, Wood | StoneWall |
| ocean | 海洋风格 | PalmWood, Glass | GlassWall |
| modern | 现代风格 | Granite, Glass, Marble | GraniteWall |

**每种风格包含**:
- primary_tiles: 主要方块列表
- primary_walls: 主要墙壁列表
- accent_tiles: 强调/装饰方块
- roof_style: 屋顶样式 (triangular/dome/flat/curved)
- roof_tiles: 屋顶方块
- furniture_style: 推荐家具
- paint_scheme: 油漆方案
- architectural_rules: 建筑规则
- biome_recommendations: 推荐生物群落
- difficulty: 难度等级

## 数据来源

- 基础数据: 手动整理，包含常用建筑元素
- 风格模板: 基于Terraria建筑社区经验总结
- NPC要求: 官方Wiki数据整理
- 房屋验证: 官方房屋判定规则
- 生物群落: 官方游戏数据

## 使用说明

### Python工具
位于 `Tools/python/` 目录:

```bash
# 查看数据库统计
python show_db_stats.py

# 初始化完整数据库
python init_full_db.py

# 补充基础方块数据
python add_basic_tiles.py

# Wiki数据爬取（需要网络）
python tile_crawler.py
```

### SQL导入
如需重建数据库，可执行SQL文件:

```bash
sqlite3 terraria_kb.db < terraria_kb_full.sql
```
